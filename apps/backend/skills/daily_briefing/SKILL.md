Daily briefing and morning summary.

## Trigger phrases

- "daily briefing"
- "morning briefing"
- "give me my briefing"
- "what's happening today"
- "morning update"

When the user asks for a briefing, use the `daily_briefing` tool. It will automatically search for current news, weather, and trending topics, then format a concise morning summary.

## Output style

Keep it conversational and concise. No markdown formatting. Structure the briefing in sections: top news, weather if relevant, and anything trending. Keep each section to 2-3 sentences.

## Routine usage

This skill works great as an automated routine. Create a cron routine with the message "run my daily briefing" and schedule it for your preferred morning time (e.g. "0 8 * * *" for 8am daily).
