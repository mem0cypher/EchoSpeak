System health monitoring and status checks.

## Trigger phrases

- "system status"
- "health check"
- "how is the system"
- "system health"
- "check system"

These queries are handled instantly by the system monitor plugin — no LLM call needed.
The plugin collects CPU usage, memory, disk space, and uptime directly.

## What it reports

- CPU usage (percentage)
- Memory usage (used / total)
- Disk usage (used / total for root partition)
- System uptime
- Python process memory
- Current time
